Battery Management System (BMS)


  • Day-Time Working

    • In Day-Time, Solar Panel Supplies load to run home appliances smoothly. If excess units produce, it will store in the Battery for BackUp.

  • Night-Time Working

    • In Night-Time, Battery Supplies load to run home appliances without any disturbance. If Battery power low, the grid-system will help for smooth operation.
